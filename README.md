# Sentiment Analyzer Lab

## Objectives
- Learn how to use AI models for sentiment analysis.
- Practice setting up API clients and sending requests.
- Analyze text inputs for Positive/Negative/Neutral sentiment.

## **Steps Completed**
1. Installed OpenAI library.
2. Set up GitHub Personal Access Token (PAT).
3. Connected to GitHub’s AI model.
4. Sent 4 sentences for sentiment analysis.
5. Received and printed the AI’s response.
